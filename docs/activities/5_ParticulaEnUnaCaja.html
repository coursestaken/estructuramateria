# Funcion de onda para particula en una caja
def Psi_n(x, n, l):
    return np.sqrt(2/l) * np.sin(n*np.pi*x/l)

# energia de un electron en cada nivel

a) La energa asociada a cada nivel.

La energa de un electrn en el nivel $n$ esta dada por:

$$E_n = \frac{n^2 h^2}{8m l^2} \tag{1}$$

Para el caso del Bitadieno tenemos que la energa de un electrn en cada nivel es:

  • $E_1 = 1.80330E-19 \quad [\text{J}]$
  • $E_2 = 7.21320E-19 \quad [\text{J}]$
  • $E_3 = 1.62297E-18 \quad [\text{J}]$

As, la energa asociada a cada nivel es:

Nivel $n=1$:

$E_{T|n=1} = 2 E_1 = 3.60660E-19 \quad [\text{J}]$

Nivel $n=2$:

$E_{T|n=2} = 2 E_2 = 1.44264E-18 \quad [\text{J}]$

$$\rule{3cm}{0pt}\rule{10cm}{0.3pt}$$

b) La energa total en el estado fundamental $\left(E_{T_F}\right)$.

$E_{T_F} = 2E_1 + 2E_2 = 1.80330E-18 \quad [\text{J}]$

$$\rule{3cm}{0pt}\rule{10cm}{0.3pt}$$

c) La energa del primer estado exitado $\left(E_{T_{1e} }\right)$ debido a la promocin de $1e^-$ desde $n=2$ a $n=3$

$E_{T_{1e} } = 2E_1 + E_2 + E_3 = 2.70495E-18 \quad [\text{J}]$

$$\rule{3cm}{0pt}\rule{10cm}{0.3pt}$$

d) La diferencia de energa de los estados $\Delta E$.

$\Delta E = E_{T_F} - E_{T_{1e} } = 9.01651E-19 \quad [\text{J}]$

Expresando $\Delta E$ como frecuencia $\displaystyle\frac{1}{\lambda} [\text{cm}^{-1}]$:

$$E = h \nu, \tag{2}$$$$\nu \lambda = c, \tag{3}$$

despejando $\nu$ de $(2)$ y sustituyendo en $(3)$ tenemos:

$$\frac{1}{\lambda} = \frac{E}{h \cdot c}. \tag{4}$$

as:

$\Delta E_f = 45390.62 \quad [\text{cm}^{-1}]$

$$\rule{3cm}{0pt}\rule{10cm}{0.3pt}$$

e) Calcule el error con respecto a la banda de absorcin observada para el butadieno $\left(46,100 \;\text{cm}^{-1}\right)$.

El error absoluto es: $\epsilon = \left | \Delta E_{\text{experimental}} - \Delta E_f \right | = 709.38 \quad [\text{cm}^{-1}]$

El error porcentual es: $\displaystyle\delta = \left | \frac{\Delta E_{\text{experimental}} - \Delta E_f}{\Delta E_{\text{experimental}}} \right | \times 100 = 1.54\%$
